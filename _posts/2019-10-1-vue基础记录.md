**记录下Vue学习过程中，关于基础知识部分的学习。**

`v-on`:监听事件绑定，代码如下：

````html
    ... 

    <div id="box">
        <input type="button" value="按钮" v-on:click="show">
        <!-- v-on 事件绑定机制 -->
    </div>
    <script type="text/javascript" src="https://unpkg.com/vue"></script>
    <script>
        let vm = new Vue({
            el:"#box",
            data:{
                mytittle:"我是标题"
            },
            methods:{ //这个methods属性中定义了当前vue实例所有可以使用的方法
                show:function(){
                    alert("hello")
                }
            }
        })
    </script>

    ...

````
